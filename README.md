## Analyze WhatsApp chat exports to uncover

*  Messaging patterns & activity trends
*  Most active users
*  Daily / monthly timelines
*  Word cloud insights
*  Emoji usage analytics
*  Link sharing behavior

Built with **Python + NLP + Streamlit**

</br>

##  Problem Statement

Messaging apps contain rich behavioral data, but raw chat exports are unstructured and unreadable at scale.

This project converts raw WhatsApp chat logs into **structured insights and visual analytics**, helping understand communication behavior, engagement patterns, and social dynamics.

</br>

##  Key Features

*  Interactive dashboard built with Streamlit
*  Clean NLP preprocessing pipeline (regex-based parsing)
*  User-level analytics (who talks most, when, and how)
*  Word frequency + most common words
*  Word cloud visualization
*  Emoji distribution analysis
*  Timeline-based activity tracking
*  Heatmap of active hours and days

This project helps users:
- Understand communication patterns in group chats

- Discover conversation trends over time

##  How It Works

1. Upload WhatsApp chat export (.txt)
2. System parses messages using regex
3. Data is structured into tabular format
4. NLP + statistical analysis applied
5. Interactive dashboard renders insights

</br>

##  Run Locally

```bash
git clone https://github.com/jeeva-anand/WhatsApp-Chat-Analysis
cd WhatsApp-Chat-Analysis
pip install -r requirements.txt
streamlit run app.py
```

</br>

 ##  Usage Guide

1. Export a WhatsApp chat **without media**.
2. Open the Streamlit application.
3. Upload the exported `.txt` file.
4. Select **Overall Analysis** or a specific user.
5. Explore interactive charts and statistics generated by the application.
